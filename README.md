# AnimuGLU
Search for and keep a local library of anime/manga details

## Uses Jikan REST
* jikan_REST
   * local install : recommended
   * online api use : change api_url in jikan_controller.py

## Python Pip Packages
* jikanpy
* pypubsub
 
 ## How to run
 * python3 main.py
