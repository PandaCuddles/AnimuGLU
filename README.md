# AnimuGLU
Search for and keep a local library of anime/manga details

## Uses Jikan REST
* jikan_REST
   * local_install (recommended)
   * online_api (change api variable in jikan_controller.py)

## Python3 Pip Packages
* pypubsub >= 4.0.3
* jikanpy >= 4.2.2
* wxpython >= 4.1.0
* python-dateutil >= 2.8.1

## How to run
* python3 main.py
